## MODIFIED Requirements

### Requirement: PDF 資料表格匯出
使用者 SHALL 能夠匯出包含所有花費與收款資料的 PDF 報告，內容以表格為主。當旅程中有預收公費紀錄時，PDF 須包含公費相關欄位。

#### Scenario: 匯出 PDF
- **WHEN** 使用者點擊「匯出 PDF」
- **THEN** 系統產生 PDF 檔案並觸發下載，內容包含：
  1. 旅行基本資訊（標題、日期、幣別）
  2. 花費明細表格（日期、類型、標題、總金額、代墊人、分攤人）
  3. 每人收款摘要表格（成員、分攤總計、代墊總計、已收總計、餘額）

#### Scenario: PDF 支援繁體中文
- **WHEN** 使用者匯出 PDF
- **THEN** PDF 中的繁體中文字元正確顯示，不出現亂碼

#### Scenario: PDF 含公費欄位
- **WHEN** 使用者匯出 PDF，且旅程中有預收公費紀錄
- **THEN** 每人收款摘要表格新增公費欄位，餘額欄以「餘額(公費餘額)」格式顯示

#### Scenario: PDF 公費花費顯示
- **WHEN** PDF 花費明細表中包含公費類型花費
- **THEN** 代墊人欄位顯示「公費」而非成員名稱
